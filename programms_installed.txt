For fonts:
# 1. Download and Unzip
cd /tmp
wget https://github.com/ryanoasis/nerd-fonts/releases/latest/download/Meslo.zip
unzip -o Meslo.zip -d meslo-temp
# 2. Move to your Dotfiles
mkdir -p ~/dotfiles/fonts
mv meslo-temp/*.ttf ~/dotfiles/fonts/
# 3. Symlink to system and refresh cache
mkdir -p ~/.local/share/fonts
ln -snf ~/dotfiles/fonts ~/.local/share/fonts/dotfiles-fonts
fc-cache -fv
# 4. Clean up
rm -rf /tmp/Meslo.zip /tmp/meslo-temp

# Stow command for syncing up config files (by creating symlinks)
Symlinks can be created with 'ln' and create a file that has the same id with the file we want to link
(changes on the one file aplly to the other file. Technically there are the same file)

Stow manages the creation and deletion of symlinks.

Naming convensions for GNU Stow
Dotfile                   GNU Stow                
~/.config/nvim            nvim/.config/nvim
~/.cinfig/alacrity        alacrity/.config/alacrity
~/.zshrc                  zshrc/.zshrc

(we need tp create in the home directory under .config a directoty with the name of the thing we want to
configure so Stow would know what to link)

(we can see the link with 'ls -lart | grep alacrity')
